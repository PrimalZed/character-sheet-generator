Handlebars

This is an assitive utility for compiling and executing Handlebars templates.

Generating a file

After selecting a root directory, this application will scan for files:

  • Template file: first of...
    • {{ '{root}' }}/src/*.handlebars
    • {{ '{root}' }}/src/*.hbs
    • {{ '{root}' }}/handlebars/*.handlebars
    • {{ '{root}' }}/handlebars/*.hbs
    • {{ '{root}' }}/hbs/*.handlebars
    • {{ '{root}' }}/hbs/*.hbs
    • {{ '{root}' }}/*.handlebars
    • {{ '{root}' }}/*.hbs
  • Data file: first of...
    • {{ '{root}' }}/src/*.json
    • {{ '{root}' }}/handlebars/*.json
    • {{ '{root}' }}/hbs/*.json
    • {{ '{root}' }}/*.json
  • Partial directory: first directory of...
    • {{ '{root}' }}/src/partials/
    • {{ '{root}' }}/handlebars/partials/
    • {{ '{root}' }}/hbs/partials/
    • {{ '{root}' }}/partials/
  • Output File Name: first of...
    • {{ '{root}' }}/*.html

Any of these selections can be changed after choosing a root directory.

The generated file will be placed in the root directory. If a file with that name already exists, it will be overwritten.

Helpers

Helpers from the following lists can be used:

  • Built-In Helpers
  • Handlebars Helpers (except for 'fs', 'html', 'logging', and 'misc' helpers)
  • Handlebars Helper Repeat

Examples

Example handlebars templates can be found at Primal Zed's Handlebars POC.

Just download the files and select example-1, example-2, or example-3 as the root directory.

For more information about Handlebars, refer to the Handlebars Guide.
